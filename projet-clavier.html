About The Project

Description

The project started as a contract for a teacher that wanted a macro keyboard to help him animate his remote classes and it finished as a graduation project.
Three versions of this project were created and 30 keyboards of the first version has been build and distributed amoung teachers at the ETS university.

Technologies

Language
C/C++
Software & Ressources
EasyEDA, Arduino IDE

My role

On this project, we were two persons. My role was to design the circuit, source all the parts needed for the project and do the soldering.
For this project I had to integrate a USB connector and a matrix array for the switches and the LEDs.
On the second version of the keyboard, I was responsable to reduce the manifacturing cost of each unit.
On the third version, we added a display and a bluetooth connectivity feature. This modification involved adding a battery and LiPO charging circuit to the board. I also ended up helping writing the firmware of the last version.
